Plan

Planning is a crucial aspect of web development because it is when many decisions are made that affect the design, implementation, and later promotion of a web. This chapter surveys issues of web planning, starting from principles based on the Web's media characteristics and user experience.

Analysis

If you have just planned a web, a big question that should be in your mind is, "Will the web accomplish its purpose?" Even when a web already is deployed and operating, you frequently should investigate whether the web is accomplishing its planned objectives. The web analysis techniques presented in this article are intended to help you check web elements in a planned or operating web.
